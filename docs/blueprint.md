# **App Name**: NetworkWeave

## Core Features:

- Distributor Management: Store and manage distributor profiles with contact information, join date, and status.
- Genealogy Visualization: Display the network marketing hierarchy with parent-child relationships and placement positions.
- Performance Metrics: Track key performance indicators such as sales volume, recruitment numbers, and commission earnings for each distributor.
- Hierarchy Querying: Enable efficient querying of multi-level hierarchies to analyze network structure and identify top performers.
- Volume Accumulation: Calculate and display accumulated sales volume at each level of the hierarchy.
- Placement Position Tracking: Track the placement position of each distributor within the hierarchy.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to represent sophistication and structure.
- Background color: Light grey (#F0F0F0) for a clean, professional look.
- Accent color: Teal (#009688) to highlight key metrics and interactive elements.
- Body and headline font: 'Inter', a sans-serif font with a modern, objective look for clear readability.
- Use simple, professional icons to represent different metrics and hierarchy levels.
- Implement a clear, hierarchical layout to visualize the network structure.
- Use subtle animations to highlight changes in performance metrics or hierarchy positions.
